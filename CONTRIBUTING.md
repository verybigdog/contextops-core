# 🛠️ Contributing to ContextOps Core

Thanks for your interest in contributing to ContextOps Core!

This project is modular and interface-driven — so contributions can be simple and pluggable.

## 🧩 Contribution Ideas

- Add a new embedding backend (e.g. Cohere, local models)
- Add a vector DB adapter (e.g. Weaviate, Milvus)
- Improve chunking strategies
- Write unit tests for search module

## 📦 Repo Structure

- `core/` – core interfaces and logic (no framework)
- `spring-support/` – Spring adapter module
- `demo-app/` – Sample REST API

## ✅ How to Contribute

1. Fork this repo
2. Clone and create a feature branch
3. Make your changes with meaningful commits
4. Submit a pull request with clear description

## 🎯 Commit Message Format

Use Conventional Commits:
- `feat:` for features
- `fix:` for bug fixes
- `docs:` for documentation
